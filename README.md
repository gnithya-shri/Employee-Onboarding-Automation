# 🚀 ServiceNow Employee Onboarding Automation

## 📝 Project Overview
This project automates the manual lifecycle of hiring a new employee within the ServiceNow platform. It replaces inefficient emails and spreadsheets with a **structured request form**, **automated manager approvals**, and **dynamic departmental task generation**.

---

## ✨ Key Features

*   **🏗️ Custom Data Model**  
    Designed a standalone table with specialized fields tailored for HR data and onboarding tracking.
*   **📋 Service Catalog Integration**  
    Built a user-friendly **Record Producer** to streamline the HR intake process for hiring managers.
*   **⚙️ Flow Designer Logic**  
    *   **Automated Approvals:** Logic-based routing to managers for quick sign-offs.
    *   **Dynamic Tasking:** Automated creation of tasks specifically for **IT, HR, or Finance** based on the department.
    *   **Status Tracking:** Real-time updates and automated record closure upon task completion.
*   **🧼 Data Hygiene**  
    Optimized "Display" settings to ensure clean, searchable audit trails within approval records.

---

## 🛠️ Tech Stack

- **Platform:** ServiceNow (Vancouver/Washington Release)
- **Studio:** App Engine Studio (AES)
- **Logic:** Flow Designer
- **Interface:** Service Catalog & Record Producers
