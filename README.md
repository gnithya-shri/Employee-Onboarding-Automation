# Employee Onboarding Automation (ServiceNow)

## Overview
This project demonstrates an automated Employee Onboarding process built using the ServiceNow platform. It is designed to reduce manual HR effort and streamline onboarding through structured workflows, approvals, and task assignments.

## Features
- Automated onboarding request creation
- HR and Manager approval workflow
- Task assignment to IT and Admin teams
- Role-based onboarding process flow
- Status tracking for onboarding requests

## Platform Used
- ServiceNow (CSA Configuration)
- Service Catalog
- Flow Designer / Workflow Editor
- ITSM Modules

## Workflow Process
1. HR raises an employee onboarding request
2. Manager reviews and approves the request
3. System generates onboarding tasks automatically
4. IT team completes system access and setup
5. Onboarding process is marked as completed
